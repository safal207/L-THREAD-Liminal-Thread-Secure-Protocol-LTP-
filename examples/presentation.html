L-THREAD / LTP · Context-first transport

Протокол для осознанных AI-систем и экономии токенов

LTP сохраняет потоковое состояние (thread + session), эмоциональный фон и контекст, шифрует метаданные, цепляет хэши сообщений и экономит 30–60% токенов за счёт TOON-энкодинга.

Уникальные возможности

  • Потоковая непрерывность: thread_id + session_id с авто‑resume
  • Аффект и контекст-теги для эмпатичных AI
  • TOON-энкодинг: минус 30–60% токенов vs JSON

Безопасность по умолчанию

  • AES-GCM шифрование метаданных (thread/session/timestamp)
  • HMAC-маршрутизация + HMAC-н�нсы, hash chaining
  • Аутентифицированный ECDH (подпись ключей, проверка давности)

Мульти-SDK & тесты

  • JS/TS, Python, Rust, Elixir — единый протокол
  • Кросс-SDK security suite (JS↔Python) и unit-тесты
  • Контейнерные команды для Rust/Elixir тестов

Бизнес-эффект

  • Низкие токенные расходы для LLM API
  • Сохранение контекста и эмоций → более “живые” ассистенты
  • Готовность к enterprise: шифрование, подписи, маршрутизация
