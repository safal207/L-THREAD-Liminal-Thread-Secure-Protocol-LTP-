# LTP Glossary (Two-Layer Terminology)

| Primary Term | Also Known As | Meaning |
|--------------|---------------|---------|
| Orientation | Liminal Position | Current semantic state of the system or agent. |
| Route Response | Meaning Branches | Enumerated plausible paths with rationales based on the present orientation. |
| Focus Snapshot | Diagnostics Telemetry | Captured state details for debugging, observability, and replay. |

Use the primary names in code and APIs. Alternate names are acceptable in explanatory docs to keep continuity with earlier drafts.
