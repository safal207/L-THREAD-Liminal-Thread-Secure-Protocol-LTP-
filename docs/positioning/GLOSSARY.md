# LTP Business Glossary

- **Turtle Orientation → State Orientation**
  The agent’s consistent alignment to the current, verified system state at each step.

- **Future Weave → Forward Consistency**
  The assurance that next-step decisions remain consistent with approved policy and prior state transitions.

- **Drift → Behavior Deviation**
  Measurable divergence between actual agent behavior and approved or expected decision paths.

- **Golden Traces → Approved Decision Paths**
  Pre-validated, regulator-acceptable execution traces used as compliance baselines.
