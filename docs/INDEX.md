## One sentence
LTP does not decide outcomes. It validates **admissible transitions** to preserve continuity over time.

## Rule of thumb
If your system asks:
- “What should we do?” → not LTP  
- “How do we do it?” → not LTP  
- “Are we still the same system over time?” → LTP
