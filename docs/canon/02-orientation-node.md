# Act II — Orientation as a Living Structure

## Key Image
```
Orientation Node
- identity (кто я сейчас)
- focus momentum (вектор внимания / инерция фокуса)
- drift history (как меня качало)
- constraints (что нельзя)
- admissible futures (что возможно)
```

## Thesis
Not a state. Not a memory.
A living orientation.

## Engineering Formula
Orientation ≠ decision; Orientation ≠ policy; Orientation ≠ reward. It is an internal axis.
